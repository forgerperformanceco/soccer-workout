# research/ — the soccer brain's source papers

Peer-reviewed papers the owner provided to ground MatchFit's training model.
Unlike `sources/` (a git-ignored drop-zone for raw ebook/course transcripts),
these academic PDFs **are committed** so the evidence base travels with the repo.

The synthesis — what each paper means for the app — lives in
[`../SOCCER-TRAINING-REFERENCE.md`](../SOCCER-TRAINING-REFERENCE.md). Read that
first; open a PDF only when you need the primary detail.

| File | Paper | Feeds |
|---|---|---|
| `Thapa-2025-jump-training-soccer-scoping-review.pdf` | Jump-training adaptations in soccer players (scoping review, Hum Mov 2025) | Speed & Power day, jump test |
| `Rodriguez-2025-core-stability-soccer-meta-analysis.pdf` | Core-stability training & soccer performance (meta-analysis, 37 studies) | 🌀 rotational/core work, agility framing |
| `Hammami-2018-soccer-training-adolescent-fitness.pdf` | Soccer training & fitness in adolescent boys (J Sport Health Sci) | Rest/Play days, 20-week horizon |

To add more: drop the PDF here, then update `SOCCER-TRAINING-REFERENCE.md` with
the synthesis and citation. Growth/maturation and concurrent-training papers are
the top gaps (see that doc's "Open questions").
